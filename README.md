# pyunz

## usage

```
usage: pyunz.py [-h] [-x EXTRACT] [-v]

optional arguments:
  -h, --help            show this help message and exit
  -x EXTRACT, --extract EXTRACT
                        extract the package automatically
  -v, --version         print version
```

## support

- tar.gz
- tar.xz
- tar.bz
- zip
- 7z
